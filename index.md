# Privacy Policy

Last updated: March 2026

This Chrome extension allows users to extract and analyze publicly visible information from webpages at the user's request.

## Data Collection
The extension does not collect personal information or track users. All processing occurs locally within the user's browser and only when the user activates the extension.

## Permissions Usage
The extension may access webpage content, browser tabs, cookies, and navigation information strictly to perform features requested by the user.

## Data Storage
User preferences may be stored locally using Chrome storage APIs. No personal data is transmitted to external servers.

## Data Sharing
No user data is sold, shared, or transferred to third parties.

## Remote Code
All executable code is packaged within the extension. No remote code execution occurs.

## User Control
The extension runs only after explicit user interaction and can be disabled or removed at any time.
